1. JavaScript中常用内置对象都有哪些？
答：
Array 数组对象；
Boolean 布尔值包装类对象；
Date 日期时间；
Math 数学对象；
String 字符串对象；
Number 数值对象； 
RegExp 正则表达式对象；
 

2. 如何创建一个数组对象方式有几种
答：
1)创建一个空数组： var a= new Array(); 
2)创建一个数组单元为5个的数组： a = new Array(5);  
3)创建一个指定数组单元的数组： a = new Array(10,20,30);
4)快捷定义数组： a=['a','b','c','d'];
