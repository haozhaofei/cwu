
   1. 什么是JavaScript？
		JS是一种脚本编程语言，解释性语言，通常直接被嵌入HTML页面
		它基于对象，具有跨平台性

   2. 在HTML中嵌入JavaScript的方法有几种？
	三种
	一使用script标签
	二超链接
	三外部导入（其标签内不要写代码）

   3. JavaScript的常用类型都有哪些？
	underfined
	null
	boolean
	number
	string
	object

   4. 常用的JavaScript的类型转换函数有那两个？
	parseInt强转整数
	parseFloat强转浮点

   5. 我们使用哪个函数判断是否不是一个整数？
	isNAN()

   6. JavaScript语言的运算符都有哪些？
	
	1. 一元运算符	++ --	一元加法和一元减法
	2. 逻辑运算符
		逻辑 NOT ! 运算符  非
		逻辑 AND && 运算符 与
		逻辑 OR  || 运算符 或
	3. 乘性运算符：*（ 乘） /(除) %（取模）求余
	4. 加性运算符： + - 
	5. 关系运算符 >  >=  < <=
	6. 等性运算符 ==  === != !==
	7. 条件运算符  ? : (三元运算符)
	8. 赋值运算符 =  +=  -= *= /= %= 
	9 逗号运算符 在一条语句中执行多个运算。
